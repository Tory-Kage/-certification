Задача: Написать программу, которая из имеющегося массива строк формирует массив из строк, длина которых меньше либо равна 3 символа. Пример:
["привет", "2", "мир", ":-)"] -> ["2", ":-)"]
["1234", "1567", "-2", "информатика"] -> ["-2"]
["Россия", "Дания", "Казань"] -> []
